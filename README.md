앱 설치법

1.우선 아래에 링크에 들어가서 각 운영체제 맞는 flutter sdk를 설치합니다.  
https://docs.flutter.dev/get-started/install


2. 설치 후 이 레포지토리를 git clone 합니다.  

3. 핸드폰 기기와 컴퓨터를 usb로 연결합니다.  

4. 다음 명령어를 순서대로 입력합니다.  

cd chalim  
flutter install
